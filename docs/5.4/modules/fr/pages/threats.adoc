== Détection des menaces sur les réseaux

{product-name} L'inspection approfondie des paquets peut être utilisée pour inspecter les paquets et les données utiles du réseau afin de détecter des attaques telles que celles figurant dans le Top 10 de l'OWASP et celles couramment utilisées dans les pare-feux pour applications web (WAF).

=== Signatures OWASP

Les capteurs DLP peuvent être créés pour détecter les attaques OWASP à l'aide des exemples de modèles suivants. Comme toujours, il peut être nécessaire de les adapter à votre environnement et à vos applications.

[,shell]
----

/cgi-bin/cgictl.*scriptName=.*[?&]scriptName=[^&]*?([\x22\x27\x3c\x3e\x28\x29]|script|onload|src)

=== Détection intégrée des menaces

{product-name} intègre également la détection d'autres menaces réseau telles que les attaques par injection SQL, les attaques DDoS (par exemple Ping Death) et les attaques par tunneling. Pour les attaques par injection SQL, {product-name} inspecte la connexion réseau (protocole SQL) entre le front-end et le pod de la base de données SQL, réduisant ainsi les faux positifs et augmentant la précision.
