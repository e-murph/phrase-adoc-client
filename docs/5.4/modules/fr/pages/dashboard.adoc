= Aperçu de l'état de la sécurité
:page-opendocs-origin: /15.ui_extension/01.dashboard/01.dashboard.md
:page-opendocs-slug:  /ui_extension/dashboard

Dans l'extension de l'interface utilisateur du portail Rancher, le **tableau de bord de NeuVector** fonctionne de la même manière que sa page indépendante. Il fournit des facteurs d'évaluation du risque de sécurité pour permettre aux utilisateurs de connaître l'état de la sécurité de leurs clusters. Dans le panneau inférieur, quatre onglets permettent aux utilisateurs de modifier les graphiques pour surveiller différentes catégories.

Dans l'onglet **Événements de sécurité**, le diagramme linéaire affiche le nombre d'événements basés sur la date dans une ligne de temps. Le diagramme à barres des événements de sécurité les plus importants affiche les 5 points d'extrémité les plus importants en termes de source et de destination.

image:dashboard_security_events.png[Tableau de bord]

Dans l'onglet **Exposition en entrée et en sortie**, le diagramme à barres empilées indique le nombre de points d'extrémité exposés par action. Sur le côté droit, les tableaux d'exposition montrent les services d'extrémité de sortie et d'entrée avec les informations d'analyse.

image:dashboard_exposure.png[Tableau de bord]

Un clic sur une ligne du tableau egress/ingress permet d'afficher les conversations sur des services spécifiques. Le nom FQDN / l'adresse IP est affiché avec un drapeau national basé sur la localisation géographique dans le champ **Hôte externe.** 

image:dashboard_conversations.png[Tableau de bord]

Un clic sur l' **hôte externe** redirige vers whois.com pour afficher les informations détaillées de l'adresse IP.

image:whois.png[Tableau de bord]

Dans l'onglet **Mode de politique**, les diagrammes circulaires montrent le ratio et le nombre de pods/services selon les différents modes de politique.

image:dashboard_policy_mode.png[Tableau de bord]

Dans l'onglet " **Actifs les plus vulnérables"**, les diagrammes à barres montrent les 5 premiers pods/nœuds qui contiennent des vulnérabilités de gravité élevée ou moyenne.

image:dashboard_top_vul_assets.png[Tableau de bord]

== Fonctions non incluses lors de la comparaison avec le NeuVector Standalone Page

* Dialogue sur l'amélioration du score.
* Protocoles d'application détectés et protégés par les règles du réseau de la couche 7.
* Rapports PDF basés sur l'espace de noms.
