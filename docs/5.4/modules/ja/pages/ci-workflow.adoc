== {product-name} CI/CDワークフローへの統合

{product-name} CI/CDプロセス全体をサポートし、ワークフローに簡単に統合して、開発および展開プロセス全体を通じてセキュリティを提供できます。

=== 開発と継続的インテグレーションのワークフロー

{product-name} 開発および CI プロセスに統合して、脆弱性スキャンを自動化できます。の使用 {product-name} Jenkins プラグインとレジストリースキャン機能により、このフェーズでイメージスキャンが可能になります。

image:ci_workflow.png[統合]

さらに {product-name} 自動アプリケーションテスト中に使用して、ネットワーク接続とコンテナの動作を分析し、ステージングまたは本番環境の問題を予測できます。

=== 継続的デプロイと本番環境のワークフロー

{product-name} 本番環境だけでなく、本番環境でも展開前のコンプライアンス テストとセキュリティ監査を実施できます。

image:cd_workflow.png[生産]

マルチベクトルセキュリティプラットフォームは、ネットワークセキュリティ、コンテナインスペクション、ホストセキュリティを組み合わせて、実行時にコンテナを保護できます。
