= Résultat de l'analyse de sécurité des actifs
:revdate1: 2025-01-16
:page-opendocs-origin: /15.ui_extension/02.runtime_security/02.runtime_security.md
:page-opendocs-slug:  /ui_extension/runtime_security

Comme Rancher Manager possède son propre onglet **Nodes** and **Workloads** pour gérer et surveiller les statuts des actifs, NeuVector injecte des composants pour afficher le résultat de son scan en cours d'exécution.

== Nœuds

Dans le tableau de l'onglet **Nœuds**, la colonne **Vulnérabilités** est injectée pour afficher les vulnérabilités détectées de chaque nœud par ordre de gravité élevée/moyenne.

image:nodes.png[Nœuds]

En cliquant sur la cellule **Vulnérabilités**, une nouvelle page d'affichage s'ouvre pour présenter les détails des résultats de l'analyse. La version de la base de données de l'analyseur est affichée, ainsi qu'une liste filtrable des vulnérabilités.

image:nodes_vul.png[Nœuds]

Un clic sur la cellule **Nom** du tableau **Nœuds** redirige vers la page de détail du nœud. NeuVector injecte un onglet **Vulnérabilités** pour afficher un tableau des vulnérabilités répertoriées.

image:node_detail_vul.png[Nœuds]

En cliquant sur le lien CVE dans la cellule **Nom** du tableau des **vulnérabilités**, une fenêtre contextuelle s'ouvre pour afficher les descriptions CVE avec un lien externe dans l'en-tête vers le site Web d'information CVE de SUSE.

image:node_detail_vul_cve.png[Nœuds]

image:suse_cve.png[SUSE]

Un bouton de téléchargement du fichier CSV est fourni au-dessus du tableau des **vulnérabilités**. Les données du fichier comprennent tous les champs figurant dans le tableau.

image:download_vul.png[CSV]

image:cve_csv.png[CSV]

== Charges de travail

Temporairement, la colonne **Vulnérabilités** ne sera pas injectée dans la table **Charges de travail** dans la version v5.4 en raison d'un problème potentiel de performance. L'équipe de NeuVector travaillera avec l'équipe de Rancher pour l'intégrer prochainement.

Cliquer sur un **nom** dans le tableau des **pods** redirige vers la page de détail du pod. NeuVector injecte un onglet **Vulnérabilités** pour afficher un tableau des vulnérabilités répertoriées.

image:pods.png[Cosse]

image:pod_detail_vul.png[Pod_Detail]
