= Ergebnis der Sicherheitsüberprüfung von Assets
:revdate1: 2025-01-16
:page-opendocs-origin: /15.ui_extension/02.runtime_security/02.runtime_security.md
:page-opendocs-slug:  /ui_extension/runtime_security

Da der Rancher Manager über eine eigene Registerkarte **"Knoten** und **Arbeitslasten** " verfügt, um den Status der Assets zu verwalten und zu überwachen, injiziert NeuVector Komponenten, um das Ergebnis der Laufzeitprüfung anzuzeigen.

== Knotenpunkte

In der Tabellenansicht der Registerkarte **"Knoten"** wird die Spalte " **Schwachstellen"** eingefügt, um die Anzahl der erkannten Schwachstellen jedes Knotens nach hohem/mittlerem Schweregrad anzuzeigen.

image:nodes.png[Knotenpunkte]

Wenn Sie auf die Zelle **Schwachstellen** klicken, wird eine neue Seite mit den Details der Scanergebnisse angezeigt. Die Datenbankversion des Scanners wird ebenso angezeigt wie eine filterbare Liste der Sicherheitsrisiken.

image:nodes_vul.png[Knotenpunkte]

Wenn Sie in der Tabelle **Knoten** auf die Zelle **Name** klicken, werden Sie auf die Detailseite des Knotens weitergeleitet. NeuVector fügt eine Registerkarte " **Schwachstellen"** ein, um eine Tabelle mit aufgelisteten Schwachstellen anzuzeigen.

image:node_detail_vul.png[Knotenpunkte]

Wenn Sie auf den CVE-Link in der Zelle **Name** in der Tabelle der **Sicherheitsrisiken** klicken, wird ein Popup-Fenster mit CVE-Beschreibungen und einem externen Link in der Kopfzeile zur CVE-Informationswebsite von SUSE angezeigt.

image:node_detail_vul_cve.png[Knotenpunkte]

image:suse_cve.png[SUSE]

Über der Tabelle der **Schwachstellen** befindet sich eine Schaltfläche zum Herunterladen einer CSV-Datei. Die Dateidaten enthalten alle Felder, die in der Tabelle enthalten sind.

image:download_vul.png[CSV]

image:cve_csv.png[CSV]

== Workloads

Vorübergehend wird die Spalte " **Schwachstellen"** in der Tabelle " **Workloads"** in der Version v5.4 aufgrund möglicher Leistungsprobleme nicht eingefügt. Das NeuVector-Team wird mit dem Rancher-Team zusammenarbeiten, um es bald zu integrieren.

Wenn Sie in der Tabelle **Pods** auf einen **Namen** klicken, werden Sie zur Detailseite des Pods weitergeleitet. NeuVector fügt eine Registerkarte " **Schwachstellen"** ein, um eine Tabelle mit aufgelisteten Schwachstellen anzuzeigen.

image:pods.png[Hülsen]

image:pod_detail_vul.png[Pod_Detail]
